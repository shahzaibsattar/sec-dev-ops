Home Component

Explanation of workings

Test data is stored in the "data service".

The home component is host to two sub components (List and Detail).

The list component subscribes to the data via the data service and fills out the list of cars.

Selecting a car sets the currentcar in the data services

The details component subscibes to the current car from the data service and is updated with the currently selected cars data each time the user selects a new car in the list.

When a data field for a car is modified the save button will become active to allow saving of that data.
